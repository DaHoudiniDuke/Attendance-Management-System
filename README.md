# Attendance Management System

A **console-based Java application** that manages student attendance using Object-Oriented Programming principles. The system allows users to register students, record attendance, search for students by ID, and generate attendance reports with calculated percentages.

---

## Description

This project was developed as part of a group programming assignment to demonstrate core **Java programming and Object-Oriented Design concepts**.

The system organizes functionality into separate classes such as:

- **Student** – stores student information
- **AttendanceRecord** – tracks attendance data
- **AttendanceManager** – manages the collection of students and attendance operations
- **Flash (Main Class)** – runs the user interface and menu system

The application simulates a simple attendance management workflow that might be used in small academic environments.

---

## Features

- Add new students to the system  
- Record attendance (Present / Absent)  
- Search for students by ID  
- Display attendance reports for all students  
- Automatically calculate attendance percentages  
- Menu-driven console interface

## Installation

1. Clone the repository:

git clone https://github.com/DaHoudiniDuke/Attendance-Management-System.git

2. Open the project in **IntelliJ IDEA** or any Java IDE.

3. Ensure you have **Java JDK installed**.

4. Run the program from:

Flash.java

## Usage

When the program starts, a menu will appear:

Add Student

Record Attendance

Search Student

Display Attendance Report

Exit

Users interact with the system by entering the number corresponding to the desired action.

Example:

Enter your choice: 1
Enter student name: John Smith
Enter student ID: 1023
